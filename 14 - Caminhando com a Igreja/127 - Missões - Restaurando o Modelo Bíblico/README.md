# 127 - Missões: Restaurando o Modelo Bíblico

Aqui compreenderemos melhor como surgiu o atual modelo missionário e como devemos trabalhar para voltar ao velho modelo bíblico. Seremos apresentados aos testemunhos de alguns homens que Deus despertou para a necessidade de ir até aos confins da terra, pregando o evangelho a toda criatura. Vimos, ainda, como a Igreja abandonou sua vocação de pregar o Evangelho, indo por todo o mundo, a todas as nações, para levar o evangelho a toda criatura, prática que foi sendo substituída pelas chamadas organizações missionárias. Na lição de hoje procuraremos entender melhor como surgiu o atual modelo missionário e quais são os problemas relacionados a ele.
