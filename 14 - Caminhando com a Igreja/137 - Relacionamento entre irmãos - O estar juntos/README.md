# 137 - Relacionamento entre irmãos - O estar juntos

Este estudo visa abordar a profundidade e importância de estarmos juntos como comunidade cristã. Tendo o amor fraternal como fundamento, exploraremos como essa interação não somente enriquece e fortalece os relacionamentos, como também contribui com o Propósito Eterno de Deus. O estar juntos é fundamental para a cooperação na obra e no Reino de Deus. No contexto dos relacionamentos entre irmãos, é algo que naturalmente gostamos de praticar.
