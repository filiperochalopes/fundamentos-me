# 109 - Compromisso para casamento

Compromisso para o casamento Nesta centésima nona lição do Fundamentos, vamos falar sobre os passos que são necessários e importantes antes de se assumir o compromisso para um casamento. Antes da tomada dessa decisão, existem algumas etapas que precisam acontecer e não podem ser negligenciadas pelos jovens e solteiros que desejam se casar. Temos aprendido ao longo dos anos que tudo que fazemos só terá valor eterno à medida que cooperar com o Propósito Eterno de Deus. Todas as verdades que aprendemos até aqui sobre a família comprovam essa afirmação.
