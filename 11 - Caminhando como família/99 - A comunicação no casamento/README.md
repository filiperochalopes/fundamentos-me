# 99 - A comunicação no casamento 

Aprenderemos quão imprescindível é uma boa comunicação para o sucesso do casamento. Seremos desafiados a enfrentar as diferenças nas formas de se comunicar, inerentes ao homem e a mulher, para aprendermos a desenvolver uma boa e exitosa comunicação no casamento. Está comunicado, está dado o bom dia para o outro. Essas formas diferentes de comunicar são muito preciosas, mas nós todos precisamos aprender, de fato, a nos comunicar uns com os outros, sobretudo com nossos filhos.
