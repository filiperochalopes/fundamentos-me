# 95 - O casamento 

Na lição anterior, vimos que a família é um projeto de Deus e tem como objetivo cooperar com o seu eterno propósito. E, assim como a família, o casamento não foi estabelecido por uma lei humana, nem inventado por alguma civilização. Ele antecede a cultura, a tradição, povo ou nação. É a partir do casamento que se estabelece a família.
