# 45 - O Espírito Santo é Deus 

Nos deteremos no que as Escrituras falam acerca dele, que é uma pessoa com quem podemos nos relacionar. Verdades fundamentais têm sido alvo de todo tipo de ataque. Creio que todos os irmãos puderam perceber como buscamos deixar claro quem é Jesus e a obra que realizou. Assim fizemos, pois queremos cooperar com Deus para que a igreja retorne aos fundamentos estabelecidos pelos apóstolos.
