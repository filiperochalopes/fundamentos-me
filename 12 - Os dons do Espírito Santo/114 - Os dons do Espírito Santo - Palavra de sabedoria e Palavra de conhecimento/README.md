# 114 - Os dons do Espírito Santo - Palavra de sabedoria e Palavra de conhecimento

Aprenderemos que a Palavra de Sabedoria é a aplicação sobrenatural do conhecimento dado por Deus. Já a Palavra de Conhecimento é o meio usado por Deus para revelar acontecimentos, situações e pensamentos do passado, presente ou futuro. Para refletir sobre esses dons, Palavra de Sabedoria e Palavra de Conhecimento, serão utilizadas passagens bíblicas nas quais eles são manifestos. Essa palavra também demonstra que os dons não são nossos, são manifestações do Espírito Santo por meio de nós.
