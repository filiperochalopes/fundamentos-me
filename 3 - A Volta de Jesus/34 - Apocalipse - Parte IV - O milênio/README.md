# 34 - Apocalipse - Parte IV - O milênio 

Apocalipse - Parte IV O Milênio Nesta trigésima quarta lição, vamos estudar um tema que não está abordado com muita clareza no sermão de Mateus 24 sobre os últimos tempos, contudo é importante entender bem. Exatamente por isso precisamos estudar e entender o que aparece relatado na Bíblia, seu significado e as implicações práticas. É importante estudar pela quantidade de detalhes que aparecem na Bíblia, porque para Deus é relevante - e não pode deixar de ser para nós. Além disso, há vários textos que falam sobre o tema, tanto no Velho Testamento quanto no Novo Testamento.
